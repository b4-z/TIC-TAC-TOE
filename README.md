# Multiplayer Game (Learning Project)

This is a real-time multiplayer web application built as a learning project to improve my full-stack development skills.

The project focuses on authentication, real-time communication, and backend architecture using Node.js and Socket.IO.

---

## 🚀 Features

- User authentication (JWT)
- Real-time multiplayer gameplay
- Room-based system for players
- Socket.IO communication between clients and server
- MongoDB database integration
- Environment variable security using `.env`

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- Socket.IO
- MongoDB
- JWT Authentication
- HTML / Talwind CSS / JavaScript (Frontend)

---

## 📝 Development Notes

The Easy Bot, multiplayer functionality, UI, backend, and the rest of this project were developed by me.

The Hard Bot uses the Minimax algorithm. Since I was not familiar with the algorithm at the time, I implemented it with the help of an AI coding agent to include an unbeatable difficulty level in the game.

I have kept this implementation in the project to be transparent about where AI assistance was used. Everything else reflects my own work and learning throughout the development of this project.
